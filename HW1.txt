1.Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
	1.Должно быть сопряжен ключ SSH с сайта в GIT Bash
	2.Скопируйте URL-адрес репозитория
	3.Откройте GIT Bash
	4.Перейти  в каталог где хотим разместить репозиторий 
	5.Введите git clone и вставьте URL-адрес “$ git clone https://github.com/username/name of repasitory”
2.Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
	1.Перейти  в каталог где хотим создать репозиторий
	2.git init
	3.git add .
	4.git commit –m
	5.git push 
3.Как получить на локальный компьютер изменения с github.com
	скопировать
4.Какую информацию нам дают команда : 
	•git status –  отображает состояние проиндексированных файлов
	•git log  –  показывает коммиты
	•git remote –v  – Верифицируем что удаленный репозиторий связан с нашим
